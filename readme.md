
## 介绍

该工程是使用lerna技术组织一个UI组件库

## 如何创建一个组件库

首先安装 Vue 官方提供的工具 [vue-cli 3.x](([https://github.com/vuejs/vue-cli](https://cli.vuejs.org/)))

``` bash
$ npm install -g @vue/cli
```

使用vue cli 3命令创建项目:

``` bash
$ vue create egova-demo-web
```

按提示信息创建

最后执行如下命令:

``` bash
$ vue add @egova/egova-starter-template
```
